# @markdownspace/components

These components are designed to be used on the Markdown Space platform. They may, or may not function elsewhere. You can simply import them from @markdownspace/components and use them on a page with mdx enabled

## Example Markdown

```mdx
---
mdx: true
---

import { TableOfContents } from 'https://esm.sh/@markdownspace/components'

<TableOfContents projectSlug="home" />
```
